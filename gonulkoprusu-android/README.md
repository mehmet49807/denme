# Gonul Koprusu Android

Native Android boilerplate using Kotlin and Jetpack Compose.

## Package

`com.gonulkoprusu`

## Included

- Shared cream/pastel Material 3 theme.
- Feed screen with username on the left and city-district on the right.
- Like, Sikayet, and Engelle actions; comments are intentionally absent.
- Profile screen with immutable username and hidden owner/admin fields.
- Premium screen documenting male-only packages and story posting rule.
- Static API endpoint constants aligned with the backend REST contract.

Firebase `google-services.json`, payment credentials, and production API base URLs are intentionally omitted until supplied.
