# Gonul Koprusu iOS

Native SwiftUI boilerplate for the Gonul Koprusu iOS client.

- Uses the same REST API contract as web and Android.
- Keeps hidden fields visible only in owner/admin profile contexts.
- Displays post feed with username on the left and city-district on the right.
- Provides Report and Block actions on profile/feed cards.
- Comments are intentionally absent.
